# 📰 Fake News Detection System

This project detects whether a news article is **Fake or Real** using Machine Learning and NLP.

## 🚀 Features
- Text preprocessing
- TF-IDF Vectorization
- Naive Bayes Classification
- Confidence Score
- Keyword Extraction (Explainable AI)
- Interactive UI using Streamlit

## 🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
python -m streamlit run app.py

## 📂 Dataset

Dataset is too large to upload here.

Download from Kaggle:
https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset

After downloading, extract and place:
- Fake.csv
- True.csv

inside the project folder.

## 📌 Output
- Predicts Fake or Real News
- Shows confidence score
- Displays important keywords
